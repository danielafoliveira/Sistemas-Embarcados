\begin{resumo}

O projeto consiste em construir um sistema de controle de acesso ativado por reconhecimento facial. Será possí­vel enviar os dados de acesso via rede para um banco de dados. Neste ponto de controle é apresentada a integração dos sistemas. São utilizadas threads para paralelizar ações de controle e verificação dos periféricos.

\textbf{Palavras-chave:} Controle de acesso, Raspberry Pi, OpenCV, reconhecimento facial, segurança, threads.

\end{resumo}
