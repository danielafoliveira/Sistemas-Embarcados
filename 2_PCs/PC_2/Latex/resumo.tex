\begin{resumo}

O projeto consiste em construir um sistema de controle de acesso ativado por reconhecimento facial. Será possí­vel enviar os dados de acesso via rede para um banco de dados. Como validação será confeccionada uma porta em miniatura.

\textbf{Palavras-chave:} Controle de acesso, Raspberry Pi, OpenCV, reconhecimento facial, segurança.

\end{resumo}
